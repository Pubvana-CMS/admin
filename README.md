# Pubvana Admin

Admin panel module for Pubvana CMS. Provides the admin shell — dashboard, user/group management, settings UI, and slot-based content injection for plugin extensions.

## Requirements

- PHP 8.1+
- [Flight School](https://github.com/enlivenapp/flight-school) ^0.2
- [Flight Shield](https://github.com/enlivenapp/flight-shield) ^0.1

## Installation

```bash
composer require pubvana/admin
```
